# i18n

## Источники переводов
- Core: `resources/lang/<locale>/*.php`
- Modules: `modules/*/lang/<locale>.php`
- Theme: `themes/<theme>/lang/<locale>.php` (опционально)

## LocaleResolver (приоритет)
1) `?lang=xx`
2) Cookie `laas_lang`
3) Default locale из настроек (settings -> config)

## Добавление нового языка
1) Добавить locale в `config/app.php`.
2) Создать файлы переводов в core/modules.
3) Проверить UI на `/admin/*` и `/`.
